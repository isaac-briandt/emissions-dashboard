# Emissions Monitoring Dashboard

A web-based frontend dashboard built with **Next.js and TypeScript** for monitoring industrial pollution levels.

---

## 📌 Features

- Fetches emissions data via a simulated API call
- Displays emissions readings in a structured table
- Highlights dangerous pollution levels with visual alerts
- Auto-refreshes emissions data every **10 seconds**
- Handles loading and API error states gracefully
- Formats timestamps into a human-readable format

## Tech Stack

- **React**
- **TypeScript**

---

## ▶️ Installation

```bash
npm install
npm run dev
```
